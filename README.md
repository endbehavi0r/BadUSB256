# BadUSB256
A BadUSB HID (Human Interface Device) project that can store and execute 256 scripts at a time.


## _Requirements:
* Arduino Pro Micro (Leonardo)
* SD Card breakout module
* SD Card
* 8 way dip switch
_
## Setup
```
Arduino ||| microSD module
VCC --> VCC
D15 --> SCK
D14 --> MISO
D16 --> MOSI
GND --> GND
A0 --> CS
```
